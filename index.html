Welcome to NightOS website

NightOS is a free and open-source operating system, which works even on old computers. It uses NodeJS and HTML5 technologies.

What is really NightOS ?

NightOS is a very safe and simple to use OS which permit you to work and play, to go on internet or to simply and securely install softwares. The project is hosted on Github and is supported by me (Clement Nerma) and motezazer.
